Project LEFKO
version 0.4.2
2015/02/08 開発開始

オリジナルのシューティングゲームです。
Processing2.2.1にて作成。


・動作環境
Linux32/64 Windows32/64
実行ファイルをご利用の際はOSに合ったファイルをダウンロードしてください。
Java必須です。
ダウンロードはこちら http://5000.sameha.org/lefko/


・インストール方法
ダウンロードしたzipファイルを適当な場所に解凍してください。


・アンインストール方法
フォルダごと削除してください。


・操作方法
マウスまたはキーボード入力で操作できます。デフォルトではマウス操作となっています。
自機はマウスカーソルを追って移動し、左クリックでショットを撃ちます。
ショットの長押しでチャージショットを撃つことができます。
ショットには消費エネルギーがあり、エネルギー(EN)が足りない状態では撃つことができません。
右クリックで武器を変更します。武器は3種類あり、Type1->Type2->Type3->Type1の順に変わります。
操作に使用するキーはキーコンフィグから確認、変更ができます。


・ゲームルール
敵のショットに当たるとシールド(SH)が減少します。シールド(SH)が0になるとゲームオーバーです。
敵にショットを当て、敵のシールドを0にすることで敵を倒すことができます。敵を倒すとスコアが増えます。
スコアが一定値に達する毎に、特別な敵が出現します。この敵は倒すまで消滅しません。


・武器解説
- Type1 : ツインショット / ツインレーザー
前方に2発のショットを撃ちます。最も標準的な性能のショットです。
チャージショットで前方に2本のレーザーを撃ちます。敵を貫通し、当てている間ダメージを与えつづけます。2本両方を当てると倍のダメージになります。
- Type2 : パワーショット / バーストショット
前方に1発のショットを撃ちます。威力が高く速度が速いですが、時間経過で速度が落ちていきます。
チャージショットで前方に1発のショットを撃ちます。このショットは時間経過で分裂していきます。分裂後は元のショットより威力と大きさが小さくなります。分裂後のショットのダメージなどの値の合計値は、分裂前のものより大きくなります。
- Type3 : スプレッドショット / リフレクション
前方広範囲に多数のショットを撃ちます。射程距離に限界があり、飛距離に応じて大きさが小さくなります。大きさが小さなものであれば、敵のショットを打ち返すことができます。
チャージショットで自身を中心にバリアを展開します。一定以内の大きさに限り、有効範囲内に入った敵のショットをまっすぐに反射します。大きすぎるショットは威力と速度が低下します。バリア展開の瞬間であれば、反射可能な大きさが通常より大きくなります。
