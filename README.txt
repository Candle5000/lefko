Project LEFKO
version 0.2.0
2015/02/08 開発開始

オリジナルのシューティングゲームです。
Processing3.0a5にて作成。


・動作環境
Linux32/64 Windows32/64
実行ファイルをご利用の際はOSに合ったファイルをダウンロードしてください。
Java必須です。
ダウンロードはこちら http://5000.sameha.org/lefko/


・インストール方法
ダウンロードしたzipファイルを適当な場所に解凍してください。


・アンインストール方法
フォルダごと削除してください。


・操作方法
マウスで操作できます。
自機はマウスカーソルを追って移動し、左クリックでショットを撃ちます。
ショットの長押しでチャージショットを撃つことができます。
ショットには消費エネルギーがあり、エネルギー(EN)が足りない状態では撃つことができません。
シールド(SH)が0になるとゲームオーバーです。
